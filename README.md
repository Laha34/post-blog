# Post Blog

Повноцінний веб-додаток для створення та управління постами, розроблений на Node.js, Express, MongoDB з використанням архітектури MVC.

## Можливості
- Створення, перегляд, редагування, видалення постів
- Хмарна база даних MongoDB Atlas
- Реалізація патерну MVC
- Адаптивний дизайн з Bootstrap
- Шаблонізатор EJS

## Технології
- **Backend**: Node.js, Express.js
- **База даних**: MongoDB Atlas
- **Шаблонізатор**: EJS
- **ODM**: Mongoose
- **Стилі**: Bootstrap 5

## Швидкий старт

### Передумови
- Node.js (v14 або вище)
- npm (Node Package Manager)
- Акаунт MongoDB Atlas

### Встановлення та запуск

1. **Клонуйте репозиторій**
   ```bash
   git clone https://github.com/Laha34/post-blog.git
   cd post-blog
2. Встановіть залежності
    npm install
3. Запустіть додаток
    # Режим розробки з автоматичним перезавантаженням
    npm run dev

    # Продакшен режим
    npm start
4. Відкрийте браузер
    Перейдіть за адресою: http://localhost:3000
